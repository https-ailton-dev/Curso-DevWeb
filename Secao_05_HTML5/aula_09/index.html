Inserir Links

Para criar um link em HTML, você pode usar a tag '<'a'>'.

'<' a href="https://www.exemplo.com">Clique aqui para visitar o Exemplo.com'<'/a'>'

Explicação dos atributos:

  • href: Este é o atributo obrigatório que especifica o destino do link. Pode ser um URL para uma página web, o caminho para um arquivo, ou um identificador para um elemento dentro do mesmo documento (um "link âncora").
  • Texto do link: Este é o texto que será exibido como o link no navegador. Pode ser qualquer texto que você queira exibir para o usuário.

No exemplo acima, quando o usuário clicar no texto "Clique aqui para visitar o Exemplo.com", ele será redirecionado para o site "https://www.exemplo.com".

Se você quiser que o link abra em uma nova aba/janela, você pode adicionar o atributo target="_blank"
