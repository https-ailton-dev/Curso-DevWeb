Formulários

Para criar formulários em HTML, você usa a tag '<'form'>'. Dentro desta tag, você pode adicionar vários elementos de entrada, como campos de texto, caixas de seleção, botões de opção, botões de envio etc. Aqui está um exemplo básico de um formulário de login:





Explicação dos elementos:

  • '<'form'>': Define o formulário. O atributo action especifica para onde os dados do formulário serão enviados e o atributo method especifica o método de envio (geralmente POST ou GET).
  • '<'label'>': Rótulo para o campo de entrada. O atributo for especifica qual campo de entrada o rótulo está associado.
  • '<'input'>': Elemento de entrada. O atributo type especifica o tipo de entrada (texto, senha, botão, etc). Os atributos id e name são identificadores do campo de entrada, e required torna o campo obrigatório.
  • '<'br'>': Quebra de linha para separar os elementos do formulário.
  • '<'input type="submit"''>': Botão de envio para enviar o formulário.

Neste exemplo, quando o usuário preencher o formulário e clicar no botão de envio, os dados do formulário serão enviados para a URL especificada no atributo action usando o método especificado no atributo method.

Variação do '<'input type='>'

Em HTML, a tag '<'input'>' é uma das mais versáteis, permitindo vários tipos de entrada de dados. Aqui estão os principais tipos que você pode usar:

  1. text: Cria um campo de texto para entrada de texto.
  2. password: Cria um campo de senha onde o texto digitado é ocultado.
  3. checkbox: Cria uma caixa de seleção que permite aos usuários selecionar várias opções.
  4. radio: Cria um botão de opção que permite aos usuários selecionar apenas uma opção de um grupo.
  5. submit: Cria um botão de envio para enviar um formulário.
  6. reset: Cria um botão de reset para redefinir os campos de um formulário para seus valores padrão.
  7. button: Cria um botão clicável.
  8. file: Cria um campo de seleção de arquivo que permite aos usuários enviar arquivos para o servidor.
  9. hidden: Cria um campo de entrada oculto que os usuários não podem ver.
  10. image: Cria um botão de imagem que envia as coordenadas do ponto onde o usuário clicou na imagem.
  11. number: Cria um campo de entrada para números.
  12. date: Cria um campo de entrada para datas.
  13. email: Cria um campo de entrada para endereços de e-mail.
  14. tel: Cria um campo de entrada para números de telefone.
  15. url: Cria um campo de entrada para URLs.
