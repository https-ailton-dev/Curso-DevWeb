Listas Não Ordenadas:

não ordenadas são criadas usando a tag '<'ul'>'. Cada item da lista é definido com a tag '<'li'>'. Por padrão, os itens são marcados com pontos.

  • Item 1
  • Item 2
  • Item 3

Listas Ordenadas:

Listas ordenadas são criadas usando a tag '<'ol'>'. Assim como nas listas não ordenadas, cada item é definido com a tag '<'li'>'. Os itens são numerados automaticamente.

  1. Item 1
  2. Item 2
  3. Item 3

Você pode aninhar listas dentro de outras listas, se desejar. Basta colocar a lista dentro de um item de outra lista. Por exemplo:

  • Item 1
  • Item 2
    • Subitem 1
    • Subitem 2
  • Item 3

Isso criará uma lista não ordenada onde o segundo item tem sua própria lista interna de itens.

Em HTML, o tipo de marcador (bullet) padrão para listas não ordenadas é um círculo. Além do marcador padrão de círculo em listas não ordenadas, HTML também oferece outras opções para o estilo dos marcadores. Alguns dos tipos de marcadores disponíveis são:

  • disc: Este é o marcador padrão. Ele exibe os itens da lista como círculos sólidos.
  • circle: Como disc, mas com círculos vazios.
  • square: Exibe os itens da lista como quadrados.
  • none: Remove completamente os marcadores, tornando a lista uma sequência contínua de texto.

Para listas ordenadas em HTML, você também pode especificar o tipo de marcador usando o atributo type na tag '<'ol'>. Aqui estão os tipos disponíveis:

  • type="1": Usa números (1, 2, 3, ...) como marcadores (este é o padrão).
  • type="A": Usa letras maiúsculas do alfabeto (A, B, C, ...) como marcadores.
  • type="a": Usa letras minúsculas do alfabeto (a, b, c, ...) como marcadores.
  • type="I": Usa números romanos maiúsculos (I, II, III, ...) como marcadores.
  • type="i": Usa números romanos minúsculos (i, ii, iii, ...) como marcadores.
