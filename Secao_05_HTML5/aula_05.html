HTML, você pode usar várias tags para formatar o texto. Aqui estão algumas das mais comuns:

Negrito: '<'strong'>' ou '<'b'>' - Define o texto como negrito.

Itálico: '<'em'>' ou '<'i'>' - Define o texto como itálico.

Sublinhado: '<'u'>' - Define o texto como sublinhado.

Tachado: '<'s'>' ou '<'strike'>' - Define o texto como tachado (riscado).

Superscrito: '<'sup'>' - Define o texto como sobrescrito.

Subscrito: '<'sub'>' - Define o texto como subscrito.
